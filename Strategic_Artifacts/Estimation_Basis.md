 GCHA Project - Estimation Basis

This document outlines the key assumptions used to generate the original estimates for the GCHA project tasks.

| Task ID | Original Estimate | Key Assumption |
| :--- | :--- | :--- |
| BAN-001 | 80h | We assume the nurse UI does not require a biometric login. |
| BER-001 | 120h | We assume the AI training data received from the Cairo team is 100% clean and structured. |
| CAI-001 | 40h | Assumes that the security audit framework is already pre-configured for the GCHA database. |

 General Constraints
- Estimates assume a standard 40-hour work week.
- No significant "Scope Creep" occurs during the development phase.
